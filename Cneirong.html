题目：猴子吃桃问题

猴子第一天吃了若干个桃子,当即吃了一半,还不解馋,又多吃了一个;第二天,吃剩下的桃子的一半,还不过瘾,又多吃了一个;以后每天都吃前一天剩下的一半多一个,到第10天想再吃时,只剩下一个桃子了.问第一天共吃了多少个桃子?...

思路

该问题可使用递归求解； 已知猴子每天都吃一般又多一个，到了第十天只剩1个了，那么由此可知它第九天吃的应该是（1+1）* 2个，及是第十天吃的个数加1在2，由此可知，它第n天吃的就是第（n+1）天的桃子数+1在 * 2，而已知第十天只剩一个了，由这个已知条件及刚刚所得公式就可递推求得第一天的桃子数，下面是具体代码以及运行结果。

代码：

解法一for 循环实现

				  public class Test {
                                        public static void main(String[] args) {
                                            int next = 1;            // 第10天 剩1个
                                            int prev = 0;            //表示前一天
                                            for(int day = 9;day > 0; day--){
                                            prev = (1+next)*2;   // next 表示后一天
                                            next = prev;
                                         }
                                    System.out.println(prev);
                                    }
                                }

解法二 递归实现

				 public class T {
                                        public static void main(String[] args) {
                                        System.out.println(total(1));
                                    }
                                        public static int total(int day){
                                        if(day == 10)
                                            return 1;
                                        else
                                            return (total(day+1)+1)*2;
                                        }
                                }
